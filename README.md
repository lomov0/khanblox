### For Developers

If you want to run this locally or contribute:

1. **Clone the repository**
   ```bash
   git clone https://github.com/lomov0/khanblox.git
   cd khanblox
   ```

2. **Serve the files**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Or using Node.js
   npx http-server -p 8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```
